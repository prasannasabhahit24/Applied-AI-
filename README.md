# Population Pyramid Visualization by Country

A Python data analysis project that visualizes population distribution by age and gender using a Population Pyramid chart.

## Project Overview

This project uses:

- **Pandas** for data cleaning and transformation
- **Matplotlib** for data visualization

The dataset contains male and female population counts across different age groups for a country and year.

The project converts raw population data into a clean format and generates a population pyramid.

---

## Features

- Load CSV population dataset
- Clean and transform raw data using pandas
- Separate male and female population data
- Merge cleaned datasets
- Filter by country and year
- Generate a horizontal population pyramid chart
- Analyze age-wise gender distribution
